#### Description

The `list` command displays all OTP names stored in the OTP file, sorted alphabetically. Each name is printed on a separate line.

#### Usage

```bash
aux4 otp list [--file <file>]
```

--file  OTP storage file (default: ~/.aux4.config/otp)

#### Example

```bash
aux4 otp list
```

```text
  - aws
  - github
  - myservice
```
